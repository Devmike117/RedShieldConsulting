Política de Privacidad

En TextEnhancer AI, respetamos tu privacidad y valoramos la transparencia. Esta política describe cómo tratamos tu información al utilizar nuestra aplicación web para mejorar documentos y generar archivos descargables en distintos formatos.

1. Información que recopilamos

No almacenamos archivos ni textos procesados. Todo el contenido se gestiona localmente en tu navegador y se procesa en tiempo real usando la API de Gemini de Google.

2. Uso de la API Gemini

Usamos el modelo generativo Gemini de Google para reescribir y enriquecer textos de forma léxica. Las solicitudes se envían a través de una clave API ofuscada y segura. No conservamos datos en servidores propios.

3. Procesamiento de archivos

Puedes subir archivos `.txt` o `.docx` que se procesan directamente en tu navegador. El texto mejorado puede descargarse como `.txt`, `.docx` o `.pdf`, sin que el archivo original sea enviado a servidores externos.

4. Inserción de imágenes

Nuestra herramienta ofrece sugerencias de imágenes basadas en el contenido de tus párrafos, utilizando descripciones generadas por Gemini y obtenidas de forma dinámica desde Unsplash.

5. Cookies

Solo usamos cookies funcionales para recordar tu aceptación del aviso de cookies. No utilizamos cookies de rastreo ni recopilamos analíticas.

6. Seguridad

Todo el procesamiento ocurre de forma local. No hay almacenamiento en la nube ni intercambio de datos con terceros, más allá del uso de APIs públicas y servicios como Unsplash.

7. Cambios en la política

Podemos actualizar esta política en cualquier momento. Revisa esta página regularmente para mantenerte informado.

8. Contacto

Si tienes dudas sobre esta política de privacidad, escríbenos a soporte@textenhancer.app.
